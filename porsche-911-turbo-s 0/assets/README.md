# Assets

This folder holds the media referenced by index.html. Placeholders are intentionally
omitted from the repo to keep it light and license-clean. Add your own:

images/
  hero.jpg              (1920x1080+ — hero poster)
  feature-aero.jpg
  feature-engine.jpg
  feature-interior.jpg
  experience.jpg        (used as fixed background)
  gallery-1.jpg ... gallery-4.jpg

videos/
  hero.mp4              (muted, looping, ~8-12s, H.264, < 4MB ideal)

Tip: export images as WebP/AVIF and wire up <picture> for best Lighthouse scores.
