# Porsche Premium Website — Project README

## Overview

This is a high-end, premium brand landing page inspired by Porsche's official website design language. The page is fully responsive, built with clean HTML and CSS, and uses real Porsche image assets and product links.

---

## Premium Features

| Feature | Description |
|---------|-------------|
| **Elegant typography** | Uses "Arial Narrow" with bold weight, minimal letter spacing, and clean hierarchy |
| **Dark gradient overlays** | Professional photo-to-text transitions using CSS `linear-gradient` |
| **Rounded card corners** | 27px border radius for a soft, luxury feel |
| **Consistent spacing** | 123px top / 120px bottom padding for breathing room |
| **Smooth hover effects** | Button darkens and lifts slightly on hover |
| **Responsive design** | Adapts from 3-column desktop to single-column mobile |
| **Real Porsche links** | Directs users to official Porsche Connect, Exclusive Manufaktur, and contact pages |

## Design Decisions

### Why 3 cards?

The three-card layout mirrors Porsche's own grid system on their official site. It creates symmetry and allows equal visual weight for each service offering.

### Why dark overlays?

The gradient overlay ensures text remains readable on any background image, while maintaining the dramatic, premium aesthetic associated with Porsche.

### Why rounded corners?

Modern luxury brands (Apple, Porsche, Tesla) use rounded corners to signal approachability and refinement.
